Stive si cozi

1.Stive

O stiva este o lista in cadrul careia avem acces la ultimul element inserat.

Astfel, noi putem insera elemente intr-un singur capat si doar din acelasi capat putem scoate elemente, cate unul rand pe rand.

Daca vrem sa accesam un anumit element din stiva, este necesar sa le scoatem pe toate cele care sunt "deasupra" lui pentru a putea ajunge la el.

Acest principiu de inserare se numeste LIFO(last in, first out) si presupune extragerea elementelor din lista in ordine inversa a insertiei.

Operatii

  • top()
    • ne spune care este primul element in ordinea inserarii (capatul "stang" al listei);
  • push()
    • adauga un nou element in acelasi capat al listei;
  • pop()
    • scoate ultimul element din lista (elementul dat de top()).

2.Cozi

O coada este o lista in cadrul careia avem acces doar la primul element inserat.

Spre deosebire de stive, putem insera elemente intr-un singur capat si extrage doar din celalalt cate unul rand pe rand.

Daca vrem sa accesam un anumit element din coada, este necesar sa le scoatem pe toate cele care sunt "sub" el pentru a putea ajunge la el.

Acest principiu de inserare se numeste FIFO(first in, first out) si presupune extragerea elementelor din lista in ordinea insertiei.

Operatii

  • top():
    • ne spune care este primul element in ordinea inserarii (capatul "stang" al listei);
  • push():
    • adauga un nou element in capatul "drept" al listei;
  • pop():
    • scoate primul element din lista (elementul dat de top()).
